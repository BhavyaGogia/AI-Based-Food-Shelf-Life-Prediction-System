The current `shared/schemas/product.schema.js` (v1.0) is **out of sync** with what Intern 2's backend needs for Week 4. | Field | Current Schema (v1.0) | Required by SRS (v2.0) | Impact |
|---|---|---|---|
| Product name | `name` | `productName` | 🔴 **BREAKING** — Batch denormalization stores `undefined` |

| Risk Level | ❌ Missing | `riskLevel` (LOW/MEDIUM/HIGH, nullable) | 🟠 FEFO priority score always 0 |
| Category enum | `"Pickle" \| "Spice" \| "Oil"` | `"snack" \| "juice" \| "pickle"` (lowercase) | 🟡 Intern 2 validator rejects Intern 1's data |

| `predictedExpiryTemplate` | ❌ Missing | `predictedExpiryTemplate` (String) | 🟡 Missing from public traceability page |

## 📐 Step 4 — Mongoose Product Model

Create `backend/src/models/Product.model.js` matching the v2.0 schema:

```javascript
const mongoose = require('mongoose');

const ProductSchema = new mongoose.Schema({
  productName: {
    type: String, required: true, unique: true, trim: true
  },
  sku: {
    type: String, required: true, unique: true, uppercase: true, trim: true
  },
  category: {
    type: String, required: true, enum: ['snack', 'juice', 'pickle']
  },
  unitSize: {
    type: String, required: true, trim: true
  },
  baseShelfLifeDays: {
    type: Number, required: true, min: 1
  },
  predictedShelfLifeDays: {
    type: Number, default: null
  },
  predictedExpiryTemplate: {
    type: String, default: 'Best Before {days} Days from Packing'
  },
  riskLevel: {
    type: String, default: null, enum: ['LOW', 'MEDIUM', 'HIGH', null]
  },
  isActive: {
    type: Boolean, default: true
  }
}, { timestamps: true });

module.exports = mongoose.model('Product', ProductSchema);
```

## 🛡️ Error Handling Middleware

Create `backend/src/middleware/errorHandler.js`:

```javascript
function errorHandler(err, req, res, next) {
  console.error('ERROR:', err.message);

  const statusCode = err.statusCode || 500;
  res.status(statusCode).json({
    success: false,
    error: err.message || 'Internal Server Error'
  });
}

module.exports = errorHandler;
```

Register it as the last middleware in `server.js`:
```javascript
app.use(errorHandler);
```

## 🤖 Gemini Service Integration

Create `backend/src/services/geminiService.js`:

```javascript
const { GoogleGenerativeAI } = require('@google/generative-ai');

const genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY);

async function analyseShelfLife(promptText) {
  const model = genAI.getGenerativeModel({ model: 'gemini-1.5-flash' });

  const result = await model.generateContent(promptText);
  const text = result.response.text();

  // Extract JSON from the response
  const jsonMatch = text.match(/```json\n?([\s\S]*?)\n?```/) || [null, text];
  return JSON.parse(jsonMatch[1].trim());
}

module.exports = { analyseShelfLife };
```
